# eslint-config-ktsn-vue

ESLint config for Vue.js project.

## Usage

Install with dependencies:

```bash
$ npm i -D eslint-config-ktsn-vue eslint prettier typescript
```

Add ESLint config (e.g. `.eslintrc.json`)

```json
{
  "extends": "ktsn-vue"
}
```

Run ESLint:

```bash
$ eslint /path/to/your-source
```
